Machtou Pichtou Defenders: Slip and Slide is a game made during the **Epic Game Jam 2026** in **48 hours** by a team of two programmers and one artist.
The theme of the jam was _Andean Mountain Range_, and the game was developed using the **Redox Engine** by [Sébastien de Graffenried](https://bsky.app/profile/sebdegraff.bsky.social).

## What I Learned

-   **3D rendering pipeline in Sokol**: At the start of the jam, the game engine did not support 3D rendering, so we had to take a previous example and integrate it into the current project.
-   **Billboard rendering**: Since the artist was more comfortable with 2D, we decided to create a 2D–3D game using billboards for the entities.
-   **Working on a large, unfamiliar codebase**: Although I had already made a [game](https://stowy.itch.io/la-nuit-tous-les-chats-sont-gris) using the Redox Engine, I still had a lot to learn.

You can play the game on [Itch.io](https://stowy.itch.io/machtou-pichtou-defenders-slip-and-slide).
